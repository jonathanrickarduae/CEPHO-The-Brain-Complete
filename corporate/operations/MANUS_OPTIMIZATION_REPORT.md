## Core Capabilities

### 1. Parallel Processing (Spawning)

Manus can spawn parallel subtasks for homogeneous operations, dramatically accelerating bulk work.

| Use Case | Example | Speed Gain |
|----------|---------|------------|
| Research multiple topics | Research 14 AI leaders simultaneously | 14x faster |
| Generate multiple documents | Create 10 SME profiles at once | 10x faster |
| Analyze multiple data sources | Process 20 competitor websites | 20x faster |
| Bulk file operations | Generate 50 template variations | 50x faster |

**Best Practice:** When facing 5+ similar tasks, request parallel processing explicitly.

**Trigger Phrases:**
- "Research all of these in parallel"
- "Generate these simultaneously"
- "Process these at the same time"

### 2. Deep Research Mode

Extended research capability for comprehensive information gathering.

| Feature | Description |
|---------|-------------|
| Multi-source synthesis | Combines information from multiple authoritative sources |
| Citation tracking | Maintains source attribution |
| Cross-validation | Verifies facts across sources |
| Depth control | Can go shallow (overview) or deep (comprehensive) |

**Best Practice:** Specify research depth and focus areas upfront.

**Trigger Phrases:**
- "Deep dive into..."
- "Comprehensive research on..."
- "Find everything about..."

## Optimization Strategies

### 1. Batch Similar Work

Instead of requesting items one by one, batch similar requests:

**Inefficient:**
- "Create SME profile for Finance"
- "Create SME profile for Legal"
- "Create SME profile for Strategy"

**Optimized:**
- "Create SME profiles for Finance, Legal, and Strategy in parallel"

### 2. Front-load Context

Provide all relevant context at the start of a task:

**Inefficient:**
- "Create a report"
- (later) "Make it professional"
- (later) "Add financial data"
- (later) "Use black and white colors"

**Optimized:**
- "Create a professional report with financial data using black and white color scheme"

### 3. Use Templates

Reference existing templates and frameworks:

**Inefficient:**
- "Create a new business plan from scratch"

**Optimized:**
- "Use the Business Plan Template to create a plan for [company]"

### 4. Checkpoint Strategically

Save checkpoints at logical milestones:

- After completing a major feature
- Before making risky changes
- At end of work session
- Before deployment

### 5. Leverage Knowledge Base

Manus learns preferences over time. Reinforce patterns:

- "Remember this format for future reports"
- "Always use this color scheme"
- "This is my preferred structure"
