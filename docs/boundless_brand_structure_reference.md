# Boundless Telecom Brand Guidelines Structure Reference

## Document Structure (8 Sections)

### 1. Introduction
- Purpose of the document
- Scope (reports, presentations, internal documents)
- Brand consistency and professionalism goals
- Reference to source materials (Financial Model, corporate website)
- Mandatory adherence statement

### 2. Logo Usage
- Logo placement (top of first page)
- Placement options: Top-center or top-left
- Exclusion Zone: Clear space around logo (height of "B" in logo icon)
- Do not alter, stretch, or recolor

### 3. Color Palette
| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| Primary Dark Color | #1A365D | rgb(26, 54, 93) | Primary headings (H1, H3), key elements |
| Accent Color | #7C3AED | rgb(124, 58, 237) | Secondary headings (H2), chart accents, call-to-action |
| Light Grey | #F3F4F6 | rgb(243, 244, 246) | Document backgrounds, table header fills |
| Text Black | #000000 | rgb(0, 0, 0) | Body text, standard content |
| Mid Grey | #666666 | rgb(102, 102, 102) | Table header text, secondary information |

### 4. Typography
- Official font: **Calibri** for all documents
- **Heading 1:** Calibri, 24pt, Bold, Dark Blue (primary color)
  - Used for major sections and document titles
  - Use sparingly for most important divisions
- **Heading 2:** Calibri, 18pt, Bold, Purple (accent color)
  - Used for sub-sections within major sections
  - Vibrant color breaks up text and highlights key areas
- **Heading 3:** Calibri, 14pt, Bold, Dark Blue
  - Used for smaller sub-sections or specific points of interest
  - Maintains primary brand color for consistency
- **Body Text:** Calibri, 11pt, Text Black
  - Left-aligned for maximum readability
  - Used for all standard paragraph text

### 5. Table Formatting
- **Header Row:** Light Grey (#F3F4F6) background, Mid Grey (#666666) text, 10pt, Bold
- **Body Rows:** White background, Text Black (#000000), 10pt
- **Borders:** Thin, light grey lines to separate cells
- Clean, easy to read, consistently styled

### 6. Chart Style
- Visually clean using official brand colors
- **Chart Types:** Bar charts, line charts, pie charts where appropriate
- **Colors:** Primary Dark and Accent colors for primary data series, grey shades for secondary
- **Labels:** Calibri, 10pt, Text Black
- **Gridlines:** Thin and light grey

### 7. Page Layout
- **Margins:** Moderate (1 inch or 2.54 cm) on all sides
- **Page Numbers:** Bottom right corner, "Page X of Y" format, Calibri 10pt, light grey text

### 8. File Naming Convention
- **Format:** `NN Document Name DDMonYYYY V#`
- **NN:** Two-digit sequence number (01, 02, 03)
- **Document Name:** Title Case with spaces (not underscores)
- **DDMonYYYY:** Date of creation (e.g., 17Jan2026)
- **V#:** Version number (V1, V2)
- **Example:** `01 Business Plan 17Jan2026 V1.pdf`

## Key Differences for CEPHO Adaptation

CEPHO uses:
- **Primary Color:** Black (#000000) instead of Dark Blue
- **Accent Color:** Pink Neon (#ff006e) instead of Purple
- **No Page Numbers:** User preference overrides standard
- **Professional consulting style** maintained
- **Calibri font** retained
