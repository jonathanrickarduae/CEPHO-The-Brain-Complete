# Boundless Telecom Brand and Style Guidelines

**Document Type:** Brand Guidelines  
**Version:** 1.0  
**Date:** 18 January 2026  
**Classification:** Internal Reference

---

## 1. Introduction

This document establishes the visual identity and document formatting standards for Boundless Telecom Group. These guidelines ensure brand consistency and professionalism across all reports, presentations, and internal documents.

**Scope:** All commercial, technical, legal, and investment materials produced for or by Boundless Telecom.

**Mandatory Adherence:** All documents representing Boundless Telecom must follow these guidelines without exception.

---

## 2. Logo Usage

### Placement

The Boundless Telecom logo should appear at the top of the first page of all documents.

**Placement Options:**
- Top center (preferred for cover pages)
- Top left (preferred for internal documents)

### Exclusion Zone

Maintain clear space around the logo equal to the height of the "B" in the logo icon. This ensures the logo remains visually distinct and uncluttered.

### Logo Integrity

- Do not alter the logo proportions
- Do not stretch or compress the logo
- Do not recolor the logo
- Do not add effects such as shadows or gradients
- Do not place the logo on busy backgrounds

## 3. Color Palette

| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| Primary Dark Blue | #1A365D | rgb(26, 54, 93) | Primary headings (H1, H3), key elements, logo background |
| Accent Purple | #7C3AED | rgb(124, 58, 237) | Secondary headings (H2), chart accents, calls to action |
| Light Grey | #F3F4F6 | rgb(243, 244, 246) | Document backgrounds, table header fills |
| Text Black | #000000 | rgb(0, 0, 0) | Body text, standard content |
| Mid Grey | #666666 | rgb(102, 102, 102) | Table header text, secondary information |
| White | #FFFFFF | rgb(255, 255, 255) | Table body backgrounds, page backgrounds |

### Color Application

**Primary Dark Blue (#1A365D):** Reserved for the most important elements including document titles, major section headings, and brand identity elements. This color establishes authority and professionalism.

**Accent Purple (#7C3AED):** Used to create visual interest and highlight secondary information. Applied to sub-section headings and interactive elements. This vibrant color breaks up text and draws attention to key areas.

**Light Grey (#F3F4F6):** Provides subtle background contrast for tables and sections without overwhelming the content.

**Text Black (#000000):** Standard body text color ensuring maximum readability.

**Mid Grey (#666666):** Secondary text for less prominent information such as table headers and captions.

## 4. Typography

### Font Family

**Official Font:** Calibri

Calibri is used for all Boundless Telecom documents. This font provides excellent readability across digital and print formats while maintaining a professional appearance.

### Heading Hierarchy

**Heading 1**
- Font: Calibri
- Size: 24pt
- Weight: Bold
- Color: Primary Dark Blue (#1A365D)
- Usage: Major sections and document titles
- Note: Use sparingly for the most important divisions

**Heading 2**
- Font: Calibri
- Size: 18pt
- Weight: Bold
- Color: Accent Purple (#7C3AED)
- Usage: Sub-sections within major sections
- Note: Vibrant color breaks up text and highlights key areas

**Heading 3**
- Font: Calibri
- Size: 14pt
- Weight: Bold
- Color: Primary Dark Blue (#1A365D)
- Usage: Smaller sub-sections or specific points of interest
- Note: Maintains primary brand color for consistency

**Body Text**
- Font: Calibri
- Size: 11pt
- Weight: Regular
- Color: Text Black (#000000)
- Alignment: Left aligned
- Usage: All standard paragraph text

### Line Spacing

- Body text: 1.15 line spacing
- Paragraphs: Single line space between paragraphs
- Headings: Double line space before, single line space after

## 5. Table Formatting

### Header Row

- Background: Light Grey (#F3F4F6)
- Text Color: Mid Grey (#666666)
- Font Size: 10pt
- Font Weight: Bold
- Alignment: Left aligned (text) or Center aligned (numbers)

### Body Rows

- Background: White (#FFFFFF)
- Text Color: Text Black (#000000)
- Font Size: 10pt
- Font Weight: Regular
- Alignment: Left aligned (text) or Right aligned (numbers)

### Borders

- Style: Thin solid lines
- Color: Light grey
- Application: All cell borders for clear separation

### Table Guidelines

- Keep tables clean and easy to read
- Use consistent styling across all tables in a document
- Avoid excessive decoration or color variation
- Ensure adequate cell padding for readability

---

## 6. Chart Style

### General Principles

Charts should be visually clean and use official brand colors consistently.

### Chart Types

Select appropriate chart types for the data being presented:
- Bar charts for comparisons
- Line charts for trends over time
- Pie charts for proportional data (use sparingly)

### Color Application

- Primary data series: Primary Dark Blue (#1A365D)
- Secondary data series: Accent Purple (#7C3AED)
- Tertiary data series: Grey shades
- Avoid using more than four colors in a single chart

### Labels and Text

- Font: Calibri
- Size: 10pt
- Color: Text Black (#000000)

### Gridlines

- Style: Thin
- Color: Light grey
- Reduce visual noise while maintaining reference points

---

## 7. Page Layout

### Margins

- All sides: 1 inch (2.54 cm)
- Consistent across all pages

### Page Numbers

- Position: Bottom right corner
- Format: "Page X of Y"
- Font: Calibri
- Size: 10pt
- Color: Light grey

### Header and Footer

- Header: Logo placement (first page) or document title (subsequent pages)
- Footer: Page numbers and confidentiality notice if required

## 8. File Naming Convention

### Format

`NN Document Name DDMonYYYY V#`

### Components

| Component | Description | Example |
|-----------|-------------|---------|
| NN | Two digit sequence number | 01, 02, 03 |
| Document Name | Title Case with spaces | Business Plan |
| DDMonYYYY | Date of creation | 17Jan2026 |
| V# | Version number | V1, V2 |

### Examples

- `01 Business Plan 17Jan2026 V1.pdf`
- `02 Financial Model 18Jan2026 V2.xlsx`
- `03 Investor Presentation 20Jan2026 V1.pptx`

### Guidelines

- Use Title Case for document names
- Use spaces between words (not underscores)
- Always include version number
- Update version number for each revision

## 9. Document Structure

### Cover Page

1. Logo (centered, top third of page)
2. Document title (Heading 1, centered)
3. Date
4. Confidentiality notice (if applicable)

### Table of Contents

- Include for documents exceeding 10 pages
- Use Heading 1 style for section titles
- Include page numbers

### Section Dividers

- Use horizontal rules to separate major sections
- Maintain consistent spacing before and after dividers

### Appendices

- Label as "Appendix A", "Appendix B", etc.
- Include in table of contents
- Use consistent formatting with main document

---

## 10. Writing Style

### Tone

- Professional and authoritative
- Clear and concise
- Confident without being aggressive

### Formatting

- Use bold for emphasis on key terms
- Use bullet points for lists
- Use numbered lists for sequential steps
- Avoid excessive use of italics

### Numbers

- Spell out numbers one through nine
- Use numerals for 10 and above
- Use numerals for percentages (5%)
- Use numerals for currency (USD 100)

## 11. Brand Voice

### Key Attributes

- **Innovative:** Emphasize technological advancement and forward thinking
- **Trustworthy:** Demonstrate reliability and transparency
- **Global:** Reflect international reach and capability
- **Professional:** Maintain business appropriate communication

### Terminology

- Use "Boundless Telecom" for formal references
- Use "Boundless" for subsequent references within the same document
- Avoid abbreviations unless previously defined
