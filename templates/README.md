# CEPHO.Ai Template Repository

All templates follow the CEPHO Brand Guidelines with black text, pink accents, and professional formatting.

## Directory Structure

```
templates/
├── branding/
│   └── CEPHO_BRAND_GUIDELINES_MASTER.html    # Master brand guidelines
├── reports/
│   ├── KPI_REPORT_TEMPLATE.html              # CEO KPI Scorecard with heat map
│   └── SPRINT_ACTION_LIST_TEMPLATE.html      # Sprint action tracking
├── briefs/
│   ├── DAILY_BRIEF_TEMPLATE.html             # Daily briefing format
│   └── PROJECT_GENESIS_BRIEF_TEMPLATE.html   # Project genesis documentation
├── presentations/
│   └── (Coming soon)
└── processes/
    └── CHIEF_OF_STAFF_SPRINT_PROCESS.md      # Sprint execution methodology
```

## Design Principles (Mandatory)

1. **Text Color**: Pure black (#000000) for all body text - NO grey
2. **Accent Color**: Pink (#E91E8C) for headings, borders, highlights
3. **Background**: White (#FFFFFF) for pages, light grey (#F5F5F5) for cards only
4. **Font**: Inter for digital, Calibri for print documents
5. **Heat Map Colors**:
   - Green (#10B981): 90-100 Excellent
   - Lime (#84CC16): 75-89 Good
   - Orange (#F59E0B): 60-74 Adequate
   - Dark Orange (#EA580C): 40-59 Developing
   - Red (#DC2626): 0-39 Critical

## Usage

1. Copy the appropriate template
2. Replace placeholder text in [BRACKETS]
3. Convert to PDF using WeasyPrint for distribution
4. All outputs must pass brand compliance check

## Template Naming Convention

`CEPHO_[TYPE]_[DESCRIPTION]_[DATE].pdf`

Example: `CEPHO_KPI_SCORECARD_18JAN2026.pdf`
